마로니에 — 양주 이미지 기반 검색 서비스 🍷
📍 프로젝트 소개

마로니는 양주 사진을 업로드하면 해당 양주에 대한 정보를 자동으로 검색해주는 웹 서비스입니다.
엘리스 AI 트랙 3기 팀 프로젝트로 진행되었으며, AI 담당 2명이 이미지 분류 모델과 데이터 수집 파이프라인을 개발했습니다.

본 프로젝트는
- 이미지 기반 양주 분류
- 크롤링 기반 데이터 수집
- 모델 예측을 통한 정보 매칭
을 목표로 제작되었습니다.
---
🚀 주요 기능
- 사용자가 업로드한 양주 이미지 분석
- TensorFlow 기반 이미지 분류 모델로 양주 종류 예측
- Selenium 크롤러로 수집한 데이터를 기반으로 양주 정보 제공
- 간단한 웹 서비스 형태로 결과 출력
---
